The Native SDK

  • Full source, all platforms.
  • Used to build Loom SDKs.
  • Enhance, trim, or modify Loom.

How to get it?

  • http://theengine.co/downloads
  • Unzip.
  • Prerequisites
    • CMake 2.8.9+
    • Ruby 1.8.9+
    • gem install rubyzip
    • XCode/Visual Studio
  • Build!

Building

Edit Rakefile, then... 
rake -t # List all build commands.
rake deploy:sdk
rake deploy:free_sdk # Just the desktop binaries; saves time
rake deploy:sdk[theCoolSdk] # Name the SDK, "dev" by default

IDE Support

Loom generates projects for a variety of IDEs. 
rake generate:xcode     # Desktop project.
rake generate:xcode_ios # iOS project.
rake generate:vs2010    # Visual Studio 2010
rake generate:vs2012    # Visual Studio 2012

Debugging

  • Set Rakefile to produce debug builds.
  • OSX: Use loom gdb or loom gdb compile.
  • Launch via IDE and set working dir to your Loom project.
